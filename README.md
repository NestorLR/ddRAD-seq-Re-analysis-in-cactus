# ddRAD-seq: Re-analysis in cactus
This repository documents the re-analysis for ddRAD-seq data from the phylogenetic study of [**Böhnert *et al.*** (2025)]( https://doi.org/10.1002/tax.13311): **Systematic and taxonomic revision of the genus *Austrocactus* (Cactaceae) based on morphology and genome wide SNP-data** and [**Merklinger*et al.*** (2021)](https://doi.org/10.1002/ajb2.1608): **Quaternary diversification of a columnar cactus in the driest place on earth**

ddRAD-seq raw data were obtanined at the European Nucleotide Archive (ENA)[https://www.ebi.ac.uk/ena/browser/home] under the BioProjects PRJEB79940 and PRJEB39114.

This repository consists in one Jupyter-Notebooks and several scripts: The notebook documents the processing data with ipyrad sofware, that includes the assembly and filtering of the ddRAD-seq data, as well as the phylogenetic and clustering analysis. The scripts mainly consists in the pipeline to proccesing data with stacks sofware.
